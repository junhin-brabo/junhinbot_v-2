## Clone esse projeto

```bash
> git clone https://github.com/junhin-brabo/junhin
```

## Instale o BOT:

```bash
> termux-setup-storage
> pkg install git
> pkg install nodejs
> git clone https://github.com/junhin-brabo/junhin
> cd junhin
> bash install.sh
```



### Para rodar
```bash
> npm i
> node index
```

```bash
> Para executar = pm2 start node index.js
> depois = pm2 monit
```
